# 事件流

事件流描述的是页面接收事件的顺序

  • 捕获阶段
  • 目标阶段
  • 冒泡阶段

# 注意

js只能选择其中的一个阶段：捕获或冒泡

  • 捕获
addEventListener('click'，function(){
    console.log("捕获")
},true)

  • 冒泡

    console.log("冒泡")
},false)
