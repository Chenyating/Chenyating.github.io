# 内部input变化，外部没有变化

反正每次值发送变化的时候，就$emit一下

# input的方法

  • 1、当input 获取到焦点时触发,最先触发
  focusMethod(e) {
      this.$emit('focus', e)
  }
1
2
3
  • 2、主要是用于 input type=button，当被点击时触发此事件
  clickMethod(e) {
    this.$emit('click', e)
  }
1
2
3
  • 3、input输入文字
  keydownMethod(e) {
    this.$emit('keydown', e)
  }
1
2
3
  • 4、当input的value值发生变化时就会触发，不用等到失去焦点
  inputMethod(e) {
    if (this.readonly) {
        this.$emit('input', this.currentValue);
    } else {
        this.currentValue = e.target.value;
        this.$emit('input', this.currentValue);
    }
  }

  keyupMethod(e) {
    this.$emit('keyup', e)
  }
1
2
3
  • 6、离开input时，且表单的value发生变化触发，没有发生变化就不触发；
  changeMethod(e) {
    this.$emit('change', e)
  }

  blurMethod(e) {
    this.$emit('blur', e)
  }
